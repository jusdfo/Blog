今天使用c# 处理XML遇到问题：
System.Xml.XPath.XPathException:“表达式的计算结果必须为节点集。”
代码如下：

    pattern = "//[@Name='" + FI.Name + "']";
    XmlNode xn = xmldoc.SelectSingleNode(pattern,xnsm);

原因：
	匹配表达式缺少节点通配符*

解决：
	将匹配模式改为：
	 pattern = "//*[@Name='" + FI.Name + "']";

个人理解：
	使用Xpath匹配前，编译器会
